# NAC3 brand assets -- Discord / community

Square icons for use in Discord servers, Slack workspaces, or any community
space dedicated to NAC3.

## Files

- `nac3-discord-icon-1024.png` -- 1024x1024 (preferred for retina + Discord).
- `nac3-discord-icon-512.png` -- 512x512 (Discord minimum, smaller download).

## Design

- Kanji: **契** (kei / chigiri) -- "contract, pledge, bond".
  Direct semantic hit for "Native Agent **Contract**".
- Background: Yujin cream `#F8F7F4`.
- Kanji + wordmark colour: Yujin indigo `#2D3A8C`.
- Underline accent: Yujin amber `#9B6A05`.
- Typeface: SimSun (kanji) + Noto Sans Bold (wordmark).
- Square layout designed with safe margin so the kanji + underline + "NAC3"
  remain visible after Discord's circular crop.
