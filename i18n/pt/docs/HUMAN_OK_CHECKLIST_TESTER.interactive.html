Instruções para o testador

Para cada tarefa abaixo:

1. Leia os passos a, b, c, d em ordem. 2. Execute exatamente o que o passo a diz, depois b, depois c. 3. Compare o que você vê/ouve com o "Resultado esperado" no passo d. 4.
