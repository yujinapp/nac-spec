NAC3 v2.2 -- Checklist do testador: layout visual / front-end

Para: o testador humano avaliando a aparência e sensação visual. Tempo: ~30 minutos. Pré-requisito: qualquer navegador moderno (Chrome recomendado).

Como usar esta checklist

Para cada tarefa:

1. Abra a URL indicada. 2. Observe a página com calma. Aguarde 30-60 segundos antes de julgar. 3. Compare com a lista "O que avaliar". 4. Marque [X] em OK se tudo estiver bom. 5. Marque [X] em PRECISA CORRIGIR se algo parecer ruim / quebrado / amador. 6. SEMPRE escreva pelo menos 1 frase no campo de Comentários, mesmo marcando OK -- descreva o que viu e como se sentiu. 7. Se notar algo que não está na lista de avaliação, escreva nos Comentários também. O campo de Comentários é a parte mais importante desta checklist.

Seja honesto, não educado. Um "precisa corrigir" com um comentário real é mais útil do que 10 "OK" marcados por reflexo.
